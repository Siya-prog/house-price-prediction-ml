# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project aims to predict house prices using Machine Learning (Linear Regression). It involves data preprocessing, feature selection, model training, and evaluation.

## 📂 Dataset
- Housing dataset with features like area, bedrooms, bathrooms, etc.
- Target variable: Price

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

## 🔍 Steps Covered
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training (Linear Regression)
6. Model Evaluation (MSE, R² Score)
7. Visualization

## 📊 Results
- Achieved good prediction accuracy using Linear Regression
- Evaluated using Mean Squared Error and R² Score

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Upload dataset
3. Run all cells
